# Template Pack

**ของแถมชิ้นที่ 2 ของ AI Money Mastery** — 4 template พร้อมใช้สำหรับเริ่มรับงาน / ขายสินค้า

เอกสารนี้เป็น **spec + copy พร้อมใช้** สำหรับสร้าง template ใน Canva หรือ Google Docs เอง เปิดไฟล์นี้ข้างจอ copy ข้อความไปใส่ใน tool ที่ระบุ แล้วปรับสีกับฟอนต์ตามแบรนด์ตัวเอง

> **หมายเหตุ** 4 template ข้างล่างเป็น "โครงและเนื้อหา" ที่ผ่านการทดลองใช้จริง ไม่ใช่ไฟล์ .canva ให้คุณคิดเอง — คุณต้องไปสร้างใน Canva เอง 30-60 นาที แต่ได้ผลลัพธ์ที่ดีกว่า template สำเร็จรูป เพราะปรับตามธุรกิจตัวเองได้

---

## สารบัญ

1. Portfolio Template (Canva) — สำหรับคนรับจ้างเขียน/แปล
2. Proposal Template (Google Docs) — ส่งลูกค้าที่ inbox มาถามงาน
3. Post ขายบริการบน Facebook — 3 version
4. Ebook Mini Template (Canva) — สำหรับคนขาย digital product

---

## Template 1 — Portfolio

**แพลตฟอร์มแนะนำ** Canva (ฟรี) → export PDF → ส่งลูกค้า  
**ขนาดหน้า** A4 portrait (210x297mm)  
**จำนวนหน้า** 9 หน้า (ปรับเพิ่มได้)  
**เวลาทำ** 2-3 ชั่วโมงครั้งแรก / 20 นาที update หลัง  

### วิธีเริ่มใน Canva

1. เข้า Canva → New design → Custom size → A4 (210x297mm)
2. สร้าง 9 หน้าเปล่า
3. เลือก color palette 2-3 สี
   - **สายอบอุ่น** ครีม + น้ำตาลเข้ม + ส้ม accent
   - **สายธุรกิจ** ขาว + เทาเข้ม + น้ำเงินเข้ม
   - **สายครีเอทีฟ** ดำ + ส้ม + ขาว
4. เลือกฟอนต์ 2 ตัว
   - Heading Sarabun Bold / Kanit Bold
   - Body Sarabun Regular / Prompt Regular
5. ใส่ header/footer เหมือนกันทุกหน้า (ยกเว้น cover)
   - Header ชื่อคุณ + logo (ถ้ามี)
   - Footer page number + website หรือ LINE ID

---

### หน้า 1 — Cover

**Layout**
- ชื่อคุณ — ตำแหน่งกลางบน 40-50pt
- Tagline — ใต้ชื่อ 18-22pt
- ประเภทงาน — มุมล่างซ้าย 14pt
- ปี — มุมล่างขวา

**Copy ที่ใช้**

```
[ชื่อ-นามสกุล]

รับเขียนแคปชัน FB/IG และบทความ
สำหรับแบรนด์ที่อยากขายได้จริง

Copywriting Portfolio
2026
```

**Tip** ใส่ภาพตัวเองหน้า cover หรือเปล่า — **ถ้าเขียน** งานตลาด mass หรือ B2C ที่ต้องการความ trust ใส่ **ถ้าเขียน** B2B หรือ ghostwriting ไม่ต้องใส่ก็ได้

### หน้า 2 — About Me

**Layout**
- Heading ใหญ่ซ้ายบน "เกี่ยวกับผม/ฉัน"
- รูป (ถ้ามี) ครึ่งขวาของหน้า
- ข้อความ 150-200 คำครึ่งซ้าย

**Copy ที่ใช้**

```
เกี่ยวกับผม/ฉัน

ผม/ฉัน [ชื่อ] นักเขียนที่ทำงานกับเจ้าของธุรกิจเล็ก
ส่วนใหญ่เป็นเพจขายของและคอร์สออนไลน์

ที่ผ่านมาช่วยลูกค้า [จำนวน] ราย เขียนแคปชัน 
บทความ และโฆษณา Facebook 
โดยเน้นให้ทุกชิ้นที่ส่งไปทำงานขายจริง ไม่ใช่แค่ดูดี

ผม/ฉันเชื่อว่า content ที่ดีเริ่มจากการเข้าใจลูกค้า
ของธุรกิจคุณก่อน ไม่ใช่เริ่มจาก template

ถ้าสนใจร่วมงาน ทัก LINE [ID] หรืออีเมล [email]
```

**Tip** อย่าเริ่ม About Me ด้วย "สวัสดีครับ/ค่ะ ผม/ฉันเป็น..." แบบ resume — เริ่มด้วยสิ่งที่คุณช่วยลูกค้า ไม่ใช่สิ่งที่คุณเป็น

### หน้า 3-5 — Case Study (3 เคส)

**Layout แต่ละหน้า**
- Heading "Case 1/2/3 — [ชื่อเคส]" ด้านบน
- ครึ่งซ้าย "Before" — ข้อความเดิม + ข้อมูลบริบท
- ครึ่งขวา "After" — ข้อความที่คุณเขียน
- แถบล่าง Result หรือความเปลี่ยนแปลง

**Copy ใช้ใน Case 1 (ตัวอย่าง)**

```
Case 1 — เพจคาเฟ่ในเชียงใหม่

Context
เพจขายเครื่องดื่มและเบเกอรี่ แคปชันเดิมสั้น 
ไม่ดึงเข้าร้าน ต้องการ tone ที่อบอุ่นมากขึ้น

Before (แคปชันเดิมของเพจ)
"ลาเต้เย็น 60 บาท จ้า"

After (เวอร์ชันที่ผม/ฉันเขียน)
"สายฝนสายเดียวก็มีเหตุผลดี ๆ 
ให้แวะร้าน ลาเต้เย็นแก้วเดียว 60 บาท 
ดื่มไปคิดงานไป เหมาะกับบ่ายแบบนี้ที่สุด
#caféชิลในกรุงเทพ"

สิ่งที่ทำ
- เปลี่ยน hook ให้เชื่อมกับบริบท (สายฝน)
- เพิ่มภาพการใช้งาน (ดื่มไปคิดงานไป)
- คง tone อบอุ่นแต่ไม่หวานเกิน
```

**Tip** ถ้ายังไม่มีลูกค้าจริง ใช้ "spec work" ที่ทำเองจากเพจจริง **ต้องระบุชัดว่าเป็น spec work ไม่ใช่โกหกว่าเพจนั้นจ้าง** พูดตรงว่า "นี่คืองานที่ผม/ฉันทำเพื่อโชว์ฝีมือ" คนเชื่อเพราะซื่อตรง

### หน้า 6 — บริการที่รับ

**Layout**
- Heading "บริการและผลงาน"
- List 4-5 บริการ แต่ละอันมี icon หรือหมายเลข
- คำอธิบายสั้น 2-3 บรรทัด

**Copy ที่ใช้**

```
บริการที่รับ

01. แคปชัน Facebook / Instagram
เขียนแคปชันที่ดึงคนให้หยุด scroll 
และนำไปสู่การคุยต่อ / ซื้อจริง
รับทั้งแบบรายโพสต์และแพ็ก 10 โพสต์

02. บทความ Blog
เขียนบทความให้ติด Google search 
และอ่านเป็นธรรมชาติ ไม่ใช่ SEO spam
500 - 1,500 คำ

03. Facebook Ads Copy  
copy สำหรับ Ad ที่เน้น conversion
3-5 variation ต่อ campaign

04. Email Marketing
email sequence ชุด welcome, 
nurture, และ promo
3-5 email ต่อชุด
```

**Tip** อย่าใส่บริการที่คุณไม่ถนัด/ไม่อยากรับจริง ๆ ถ้าคนจ้างเข้ามาคุณต้องทำได้ถึงระดับไหน ใส่เฉพาะที่ "ส่งได้สบาย"

### หน้า 7 — ราคาและแพ็กเกจ

**Layout**
- Heading "ราคา" 
- ตาราง 3 คอลัมน์ (บริการ / ราคา / จำนวน revision)
- Note ด้านล่างเกี่ยวกับการชำระเงิน

**Copy ที่ใช้**

```
ราคาเริ่มต้น

แคปชัน FB/IG
├─ 1 โพสต์: 200 บาท  
├─ 5 โพสต์: 900 บาท (เฉลี่ย 180/ชิ้น)
└─ 10 โพสต์: 1,700 บาท (เฉลี่ย 170/ชิ้น)

บทความ Blog
├─ 500 คำ: 600 บาท
├─ 1,000 คำ: 1,200 บาท  
└─ 1,500 คำ: 1,800 บาท

Facebook Ads Copy
├─ 3 variation: 800 บาท
└─ 5 variation: 1,300 บาท

Email Marketing
├─ 3 emails: 2,500 บาท
└─ 5 emails: 4,000 บาท

—
เงื่อนไข
- Revision 2 รอบ include ในทุกแพ็ก  
- มัดจำ 50% ก่อนเริ่มงาน
- ส่งงานภายใน 3-7 วัน ขึ้นอยู่กับขอบเขต
- ชำระผ่านโอนธนาคาร / PromptPay
```

**Tip** อย่าโชว์ราคาต่ำกว่า 50 บาท/โพสต์ ต่อให้เริ่มทำคนเดียว ลูกค้าที่จ่าย 50 มักเป็นลูกค้าที่เจ้าปัญหาสุด — ตั้งต่ำสุด 100 บาท และเสนอ discount เป็น package แทน

### หน้า 8 — Testimonial (ว่างไว้รอเติม)

**Layout**
- Heading "ลูกค้าพูดว่าอย่างไร"
- 2-3 quote card พร้อมชื่อลูกค้า + ธุรกิจ

**Copy ที่ใช้ (placeholder)**

```
ลูกค้าพูดว่าอย่างไร

"[quote 1-2 ประโยคจากลูกค้า]"
— [ชื่อลูกค้า] / [ชื่อเพจ/ธุรกิจ]

"[quote]"
— [ชื่อ] / [ธุรกิจ]

"[quote]"  
— [ชื่อ] / [ธุรกิจ]
```

**Tip** ถ้ายังไม่มีลูกค้า ทิ้งหน้านี้ไว้ก่อน **ไม่แต่ง testimonial ปลอม** — ลูกค้าอ่านออก อ่านเดือนแรกได้เคสแรก ขอ testimonial ทุกครั้ง เติมเข้ามาทีละ quote

### หน้า 9 — Contact / CTA

**Layout**
- Heading ใหญ่กลางหน้า
- Contact info 3-4 ช่องทาง
- CTA ปิด

**Copy ที่ใช้**

```
พร้อมเริ่มงานด้วยกัน

ถ้ามีงานที่อยากคุย ส่งมาได้เลย
ตอบภายใน 24 ชั่วโมง

LINE: [ID]
Email: [email]
Facebook: [link เพจ]

—

ก่อนส่ง brief ช่วยเตรียมข้อมูล 3 อย่าง
- ธุรกิจของคุณคืออะไร ลูกค้าคือใคร
- ประเภทงานและจำนวนชิ้น
- Deadline ที่ต้องการ

แล้วค่อยคุย scope และราคาต่อกัน
```

**Tip** CTA ต้องบอกขั้นตอน ไม่ใช่แค่ช่องทาง — "ถ้าส่งมาต้องเตรียมอะไร" ทำให้ลูกค้ารู้สึกว่าเป็นมืออาชีพ ไม่ใช่มือใหม่

## Template 2 — Proposal (Google Docs)

**แพลตฟอร์ม** Google Docs → share link หรือ export PDF  
**ความยาว** 2-3 หน้า  
**เวลาทำ** 15-20 นาทีต่อ proposal ครั้งถัดไป (หลังมี template นี้)

### วิธีเริ่มใน Google Docs

1. Google Docs → New → Blank document
2. ตั้งชื่อไฟล์ "Proposal Template — Copywriting Service"
3. ใส่ header (logo หรือชื่อ) + footer (page number)
4. ใช้ style "Heading 1" สำหรับหัวข้อหลัก, "Heading 2" สำหรับย่อย
5. Save เป็น template หลัก — ทุกครั้งรับ brief ใหม่ ให้ duplicate ไฟล์นี้

---

### Section 1 — Intro

```
Proposal

Project: [ชื่องาน]
Client: [ชื่อลูกค้า / บริษัท]  
Prepared by: [ชื่อคุณ]
Date: [วันที่]

—

สวัสดีครับ/ค่ะ คุณ [ชื่อลูกค้า]

ขอบคุณที่ให้โอกาสเสนองานให้ในครั้งนี้ จากที่คุยกันเมื่อวานนี้
ผม/ฉันเข้าใจว่าคุณกำลังมองหาคนช่วยเขียน [ประเภทงาน] 
สำหรับ [บริบทธุรกิจ] เพื่อ [เป้าหมายหลัก 1-2 อย่าง]

เอกสารนี้สรุป scope ราคา และ timeline ที่เสนอ 
หลังอ่านจบถ้ามีจุดไหนอยากปรับ reply กลับมาได้เลย
```

**Tip** Intro ที่ดีต้อง "summarize สิ่งที่ลูกค้าพูดมา" กลับไปให้ฟัง ลูกค้าจะรู้สึก "เขาฟังผมอยู่จริง ๆ" — อย่าเริ่มด้วย "ผม/ฉันเป็น..."

---

### Section 2 — Scope

```
ขอบเขตของงาน (Scope)

จากที่คุยกัน งานที่จะทำประกอบด้วย

1. [งานหลัก] — [จำนวน] ชิ้น
   เช่น "แคปชัน Facebook 10 โพสต์"
   - แต่ละโพสต์ความยาว 80-120 คำ  
   - ใน tone [ระบุ]
   - ภายใต้ theme [ระบุ]

2. [งานรอง ถ้ามี] — [จำนวน]

สิ่งที่อยู่ใน scope  
- ร่างแรก + 2 revision
- ส่งงานใน Google Docs
- ทำตาม brief และ brand voice guide ที่ได้รับ

สิ่งที่อยู่นอก scope (มีค่าใช้จ่ายเพิ่ม)
- ภาพประกอบ / graphic design
- การโพสต์เอง (จัด schedule ไทม์ไลน์)
- การปรับสไตล์ใหม่หลัง revision ครั้งที่ 2
- งานเพิ่มเติมที่ไม่ได้ระบุในข้อ 1-2
```

**Tip** Scope ที่ชัดจะป้องกัน "scope creep" — ลูกค้าที่เพิ่มงานเรื่อย ๆ โดยไม่เพิ่มเงิน ระบุ "นอก scope" ให้ชัดตั้งแต่แรก ปลอดภัยทั้งคู่

---

### Section 3 — Timeline

```
Timeline

Week 1
- รับ brief เต็ม + ตัวอย่าง content ที่ชอบ (Day 1)
- ส่งร่างแรก (Day 5)

Week 2
- ลูกค้า review + feedback (Day 6-7)
- ส่ง revision รอบ 1 (Day 8)
- Review + feedback (Day 9)
- ส่ง revision รอบ 2 / งานสุดท้าย (Day 10)

รวมระยะเวลา 10 วันทำการ

—

Deadline ที่ไม่สามารถรับได้
- งานเร่งด่วนภายใน 24 ชั่วโมง
- งานช่วงเทศกาลใหญ่ (ถ้ามี แจ้งล่วงหน้า 3 วัน)
```

**Tip** Timeline ที่มี buffer = ส่งเร็วได้ดูดี / ส่งช้าไม่หลุด deadline — อย่าสัญญาเร็วเกินจริง

---

### Section 4 — Pricing

```
ราคา

[งานหลัก] × [จำนวน]  ....................... [ราคา] บาท
[งานรอง ถ้ามี] × [จำนวน]  .................. [ราคา] บาท

รวมทั้งโปรเจ็กต์                           [ราคารวม] บาท

การชำระเงิน
- มัดจำ 50% ก่อนเริ่มงาน ([ครึ่งราคา] บาท)
- ชำระส่วนที่เหลือเมื่อส่งงานครบ ([ครึ่งราคา] บาท)
- ธนาคาร [ชื่อ] / เลขบัญชี [...] / ชื่อบัญชี [...]
- หรือ PromptPay [เบอร์]
```

**Tip** มัดจำ 50% ไม่ใช่เพื่อ "เงิน" อย่างเดียว แต่เป็น "สัญญาณว่าลูกค้าจริงใจ" ลูกค้าที่ไม่ยอมมัดจำ 90% หายก่อนงานเสร็จ

---

### Section 5 — T&C และ CTA

```
เงื่อนไขอื่น ๆ

- Copyright ของงานเป็นของลูกค้าหลังชำระเงินครบ
- ผม/ฉันเก็บสิทธิ์ในการใช้งานชิ้นนี้เป็น portfolio 
  (ถ้าไม่ต้องการ แจ้งได้ก่อนเริ่มงาน)
- ถ้ายกเลิกหลังเริ่มงานแล้ว มัดจำไม่คืน
- งานเสร็จส่งใน Google Docs สำหรับแก้ไข + PDF final

—

ถ้า proposal นี้ตรงกับที่คุย
reply กลับมา "approve" หรือ ส่งคำถาม/ปรับ ได้เลย

รอตอบกลับ
[ชื่อคุณ]
[email] / [LINE]
```

**Tip** CTA ต้องเรียบ — "reply คำว่า approve" เข้าใจง่ายกว่า "ตัดสินใจอย่างไร โปรดแจ้ง" ลูกค้าไม่ใช่ CEO จะใช้ภาษาทางการเกิน

✅ จ้างคนมาแล้วได้งาน generic ไม่ตรงแบรนด์

## Template 4 — Ebook Mini (Canva)

**แพลตฟอร์ม** Canva (ฟรี)  
**ขนาด** A4 หรือ US Letter (ใช้ได้ทั้งสอง) แนะนำ A4  
**จำนวนหน้า** 18 หน้า (ปก 1 + เนื้อหา 15 + CTA 2)  
**เวลาทำ** 4-6 ชั่วโมงแรก / 2-3 ชั่วโมง ebook ถัดไป

### วิธีเริ่มใน Canva

1. Canva → Custom size → 210x297mm
2. 18 หน้าเปล่า
3. ตั้ง template master (หน้าไหนใช้ layout อะไร)
4. ใช้ grid system 2 คอลัมน์ หรือ 1 คอลัมน์เต็ม
5. Save Brand kit — สี ฟอนต์ logo (Canva Pro ถ้ามี) หรือจดเป็น note ไว้

---

### โครงสร้างหน้า

```
หน้า 1       Cover
หน้า 2       Copyright + About Author
หน้า 3       สารบัญ / Introduction
หน้า 4       บทที่ 1 (4-5 หน้า)
หน้า 9       บทที่ 2 (4-5 หน้า)
หน้า 14      บทที่ 3 (3-4 หน้า)
หน้า 17      Bonus / Action Sheet
หน้า 18      CTA ซื้อ product ถัดไป / follow social
```

**Color palette แนะนำ 3 แนว**
- **Warm** ครีม #FFF8EC + น้ำตาลเข้ม #4A2C2A + ส้ม accent #E87A5D
- **Modern** ขาว #FFFFFF + ดำ #1A1A1A + เหลืองนีออน accent #F4FA58  
- **Minimal** สีเทาอ่อน #F4F1EC + เทาเข้ม #2E2E2E + เขียว olive #87956F

**Font pairing แนะนำ** (Canva ฟรี)
- **สาย Professional** Heading "Kanit Bold" / Body "Sarabun Regular"
- **สาย Modern** Heading "Prompt Bold" / Body "IBM Plex Sans Thai"
- **สาย Friendly** Heading "Mitr Bold" / Body "Kanit Regular"

---

### หน้า 1 — Cover

**Layout**
- ชื่อ ebook ตำแหน่งกลางหรือซ้ายบน 48-60pt
- Subtitle ใต้ชื่อ 20-24pt
- ชื่อผู้เขียน มุมล่างซ้าย 16pt
- Visual element 1 อัน (abstract shape หรือ icon)

**Copy placeholder**

```
[ชื่อ Ebook]

[Subtitle ที่อธิบาย value ใน 8-12 คำ]

by [ชื่อผู้เขียน]
```

**Tip** Cover ต้อง "อ่านได้ในขนาด thumbnail 200x300px" — export แล้วย่อทดสอบ ถ้าชื่อหนังสืออ่านไม่ออก ต้องขยายขนาด หรือตัดคำให้น้อยลง

### หน้า 3 — สารบัญ / Introduction

**Layout**
- ครึ่งบน: Intro 150-200 คำ
- ครึ่งล่าง: สารบัญ list

**Copy ที่ใช้**

```
Introduction

[Hook 1-2 ประโยคเริ่มต้นที่ดึงความสนใจ]

ใน ebook เล่มนี้คุณจะได้
- [benefit 1]
- [benefit 2]
- [benefit 3]

อ่านจบในเวลา [ตัวเลข] นาที
นำไปทำได้ทันทีใน [ตัวเลข] วัน

—

สารบัญ

บทที่ 1 — [ชื่อบท]                    หน้า 4
บทที่ 2 — [ชื่อบท]                    หน้า 9
บทที่ 3 — [ชื่อบท]                    หน้า 14
Bonus — [ชื่อ bonus]                  หน้า 17
```

---

### หน้า 4-16 — เนื้อหาแต่ละบท

**Layout หน้าแรกของบท**
- Heading ใหญ่ "บทที่ X" ด้านบน 36-42pt
- ชื่อบท 24-28pt
- Divider line
- Body ต่อจากนั้น

**Layout หน้าต่อของบท**
- ใช้ heading 2 (subsection) 18-22pt
- Body 11-13pt
- Pull quote ถ้ามี 14-16pt italic
- มี breathing space ระหว่าง section

**Tip** อย่าอัดตัวหนังสือเต็มหน้า ใช้ whitespace 30-40% ต่อหน้า อ่านง่ายกว่า ดูมีคุณภาพกว่า

---

### หน้า 17 — Bonus / Action Sheet

**Layout**
- Heading "Bonus — Action Sheet"
- Checklist 10-15 ข้อ
- ที่ว่างให้ลูกค้า tick หรือจด

**Copy ที่ใช้**

```
Bonus — Action Checklist

เริ่มทำทีละข้อ พอทำเสร็จ tick ช่องข้าง ๆ

□ Step 1: [action]
□ Step 2: [action]
□ Step 3: [action]
...
□ Step 15: [action]

—

Review ทุก 7 วัน

วันที่ 7:   ทำได้กี่ข้อ? ติดตรงไหน?
วันที่ 14:  ผลลัพธ์เห็นหรือยัง?
วันที่ 30:  ปรับแผนอะไรบ้าง?
```

### หน้า 18 — CTA

```
ขอบคุณที่อ่านจบ

ถ้าเนื้อหาใน ebook นี้มีประโยชน์ 
ฝาก 2 สิ่งสำหรับก้าวต่อไป

1. Follow และเข้าคอมมูนิตี้
   Facebook Group: [link]
   LINE Community: [link]

2. ถ้าต้องการเจาะลึกขึ้น
   [ชื่อ product ถัดไป + ราคา]
   [link สั่งซื้อ]

—

ให้ feedback ได้ที่
[email] / [LINE]

อ่านแล้วใช้ได้จริง = รีวิว 1 บรรทัดเป็นของขวัญที่มีค่าสุด
```

**Tip** หน้า CTA สุดท้ายห้ามขายแรง — คนเพิ่งอ่านเนื้อหามาจบ ต้องการ acknowledgement ก่อน upsell ลูกค้าจะรู้สึกว่าไม่ได้โดนใช้เพื่อหาเงินอย่างเดียว

## คำแนะนำทั่วไปก่อนเริ่ม

**1. สร้าง Brand Kit ใน Canva ก่อน (ถ้ามี Canva Pro)**  
ทุก template ใน pack นี้จะใช้สีและฟอนต์ชุดเดียวกัน ตั้งครั้งแรกใช้ซ้ำตลอด

**2. Export format ที่แนะนำ**  
- Portfolio / Proposal / Ebook → PDF (ลดขนาดไฟล์ option "standard")
- Post FB → PNG หรือ JPG (quality 80-90%)

**3. เก็บไฟล์ต้นฉบับไว้ในโฟลเดอร์เดียว**  
`starter-kit-templates/` บน Google Drive / Dropbox — จะเปิดแก้เมื่อไหร่ก็ได้

**4. ปรับให้เป็นของตัวเอง อย่า copy ตรง**  
Template นี้คือ "โครง" ไม่ใช่ "สำเร็จ" ใส่เสียงของตัวเองเข้าไป — สีที่ชอบ สัญลักษณ์ส่วนตัว เรื่องเล่าจริง

**5. อัพเดตทุก 3-6 เดือน**  
ราคา portfolio case study — ไม่ควรปล่อยให้เก่า คนจ้างเข้ามาเห็น case ล่าสุด 1 ปีที่แล้ว จะเริ่มลังเล

## คำปิดท้าย

Template คือ "โครงสำหรับเริ่ม" ไม่ใช่ "ผลลัพธ์สุดท้าย"

คนที่ใช้ template เหมือนกันจะได้ผลคนละแบบ เพราะของจริงอยู่ที่เนื้อหาที่คุณใส่เข้าไป ไม่ใช่ layout

ถ้ามีคำถามหรืออยาก share template ที่ทำเสร็จแล้ว post เข้ามาในกลุ่ม community ได้ — คนอื่นจะได้เรียนรู้จากงานของคุณด้วย
