# บทที่ 7
## AI เครื่องมืออื่น ๆ ที่ควรรู้จัก — เลือก 3 ตัวพอ

---

## 7.1 ทำไมบทนี้เริ่มด้วยคำเตือน

สมมติว่าคุณเปิด Twitter หรือ TikTok สาย AI มา 1 สัปดาห์ คุณจะเห็นโพสต์แบบนี้วันละ 10-20 ครั้ง

> "50 AI Tools ที่ทุกคนต้องรู้ในปี 2569"
> "AI Stack ของ freelancer ระดับเทพ — 23 tools ในรูปเดียว"
> "ถ้าคุณไม่รู้จัก [tool X] คุณล้าหลังแล้ว"

คนที่อ่านโพสต์พวกนี้จบ มักทำ 2 อย่าง หนึ่ง สมัคร 10-15 tools ในสัปดาห์เดียว กดไปเล่นทีละตัว 15 นาที ไม่เจาะตัวไหนจริง ๆ จ่ายเงิน subscription เดือนละ 2,000-3,000 บาทกับของที่ไม่ได้ใช้

สอง รู้สึก overwhelmed เลิกใช้ AI ทั้งหมด กลับไปทำงานแบบเดิม

บทนี้ไม่ใช่ list "50 tools ต้องรู้" บทนี้ตรงกันข้าม บทนี้ช่วยคุณเลือก 3 tools ที่ใช้จริง แล้วใช้ให้เก่ง

กฎของบทนี้ รู้ 3 tools ลึก ดีกว่ารู้ 30 tools ตื้น

## 7.2 Toolbox แบ่งตามประเภทงาน

เริ่มจาก category ก่อน คุณต้องเข้าใจว่า AI ทำงานอะไรได้บ้าง แล้วค่อยเลือก tool ในแต่ละ category

### ประเภท 1: AI คุยและเขียน (Core)

**ChatGPT** (OpenAI)
- จุดเด่น versatile ที่สุด เก่ง brainstorm code Excel formula ภาษาอังกฤษดีเยี่ยม
- จุดอ่อน ภาษาไทยบางบริบทยังแข็ง
- ราคา ฟรี limited / Plus ประมาณ 600-700 บาท/เดือน

**Claude** (Anthropic)
- จุดเด่น ภาษาไทย creative writing ลื่นสุด รับไฟล์ยาว 50-200 หน้าได้ดี
- จุดอ่อน ไม่มี image generator built-in
- ราคา ฟรี limited / Pro ประมาณ 600-700 บาท/เดือน

**Gemini** (Google)
- จุดเด่น เชื่อม Google Search ข้อมูลล่าสุด ฟรีใจดีสุด
- จุดอ่อน ความสามารถ creative ยังตามหลัง Claude และ ChatGPT
- ราคา ฟรีเป็นหลัก / Advanced ประมาณ 700 บาท/เดือน

**แนะนำสำหรับคนเริ่ม** สมัครฟรี 1 ตัว (Claude หรือ ChatGPT) ใช้ 2-3 เดือน พอเริ่มรู้สึกว่าฟรีไม่พอ ค่อยขยับเป็น paid

### ประเภท 2: AI สร้างภาพ

**Canva AI**
- จุดเด่น ในตัว Canva ใช้ต่อจาก template ได้เลย ภาพระดับกลาง
- จุดอ่อน ไม่เหมาะกับภาพที่ต้องการ photorealistic
- ราคา ฟรี limited / Pro ประมาณ 400-500 บาท/เดือน

**Ideogram**
- จุดเด่น ใส่ text บนภาพได้แม่นที่สุด เหมาะทำ poster, banner, ปก ebook
- จุดอ่อน realistic ยังไม่เท่า Midjourney
- ราคา ฟรี 20 generations/วัน / paid $8-20/เดือน

**Midjourney**
- จุดเด่น ภาพ artistic คุณภาพสูง
- จุดอ่อน ใช้ผ่าน Discord งงตอนแรก ไม่มีฟรี
- ราคา $10-60/เดือน

**Flux / Nano Banana / เครื่องมือ image ใหม่ ๆ**
- ภาพรวมตลาดของ AI ภาพเปลี่ยนเร็ว ปี 2569 มีเครื่องมือใหม่ออกทุกไตรมาส
- ก่อนสมัคร เช็คล่าสุดว่าตัวไหน dominant

**แนะนำสำหรับคนเริ่ม** Canva AI ฟรี + Ideogram ฟรี 2 ตัวคู่กันพอ ไม่ต้อง Midjourney ตอนนี้

### ประเภท 3: AI เสียง

**ElevenLabs**
- จุดเด่น voice ธรรมชาติสุดในตอนนี้ รองรับภาษาไทย clone เสียงตัวเองได้ (ต้องระวังจริยธรรม)
- จุดอ่อน free tier 10,000 ตัวอักษร/เดือนอาจไม่พอสำหรับ YouTube
- ราคา ฟรี limited / $5-22/เดือนสำหรับ commercial use

**Whisper** (OpenAI) / **MacWhisper**
- จุดเด่น ถอดเสียงเป็นข้อความแม่นมาก 99 ภาษา ภาษาไทยดีกว่า Google Speech
- ใช้ยังไง ผ่าน API, open-source หรือ MacWhisper (Mac app $25 once)

**แนะนำ** ถ้าไม่ทำ YouTube/TikTok faceless ข้าม category นี้ไปก่อน

### ประเภท 4: AI ค้นหาและ research

**Perplexity**
- จุดเด่น search เหมือน Google แต่ตอบพร้อม citation แทน link
- เหมาะกับ research niche, เช็คข้อมูลก่อนเขียนบทความ
- ราคา ฟรี / Pro $20/เดือน

**ChatGPT Search / Gemini**
- มี search built-in แต่ยังไม่ดีเท่า Perplexity ในงาน research จริงจัง

**แนะนำ** ถ้าทำ affiliate / blog Perplexity มีประโยชน์ ตัวอื่นฟรีพอแล้ว

### ประเภท 5: AI สรุปเอกสารยาว

**NotebookLM** (Google)
- จุดเด่น โยน PDF 10-50 ไฟล์ได้พร้อมกัน ถามเป็นภาษาธรรมชาติ ได้คำตอบพร้อม citation ไปยังไฟล์ต้นฉบับ
- Feature น่าสนใจ แปลงเป็น podcast audio 2 คนคุยกันเรื่องเอกสารนั้น (อังกฤษชัดสุด ไทยเริ่มรองรับ)
- ราคา ฟรี

**แนะนำ** เหมาะมากกับคนรับงานสรุป Ch 11 หรือคนเรียนรู้ topic ใหม่เร็ว ๆ

### ประเภท 6: AI วิดีโอ

**Runway**
- จุดเด่น สร้างวิดีโอจาก text หรือภาพ คุณภาพดี
- จุดอ่อน แพง (15 credits/วิดีโอสั้น)
- ราคา ฟรี limited / $15-76/เดือน

**CapCut AI**
- จุดเด่น ตัดต่อวิดีโอ + AI feature (auto caption, background remove, script-to-video)
- เหมาะกับคนทำ TikTok / Shorts
- ราคา ฟรีเยอะ / Pro $8/เดือน

**HeyGen**
- จุดเด่น avatar พูดได้ เลือกหน้าหรือ clone ของตัวเอง
- ราคา $24-89/เดือน

**แนะนำ** ถ้าเริ่ม CapCut ฟรีพอแล้ว Runway / HeyGen รอเดือน 3-6 ที่มั่นใจแล้วค่อย

### ประเภท 7: AI ดีไซน์และ presentation

**Canva (ไม่ใช่ Canva AI)**
- เครื่องมือ design ที่คนไทยใช้เยอะสุด template หลายหมื่น
- ราคา ฟรี / Pro ประมาณ 400-500 บาท/เดือน
- เป็น backbone ของหลาย workflow

**Gamma / Tome**
- สร้าง presentation จาก prompt ใช้เวลา 5-10 นาที ได้ slide 10-15 หน้า
- ราคา ฟรี limited / paid $8-15/เดือน

### ประเภท 8: Automation (ระดับ advanced)

**Zapier / Make / n8n**
- เชื่อม app เข้าด้วยกัน ทำงานอัตโนมัติ
- ตัวอย่าง ลูกค้ากรอก form → ส่ง email อัตโนมัติ → เพิ่มใน Google Sheet → แจ้ง LINE
- ราคา ฟรี limited / paid $20-100/เดือน

**แนะนำ** ข้ามไปก่อน รอเดือน 6-12 ที่มีรายได้สม่ำเสมอแล้วค่อยลงทุน

## 7.4 วิธีเลือกเมื่อมี 2-3 ตัวคล้ายกัน

ตัวเลือก tools จะเยอะเพราะเครื่องมือใหม่ออกทุกสัปดาห์ ถ้าเจอ 2-3 ตัวในหมวดเดียวกัน ถามตัวเอง 3 คำถาม

### คำถาม 1: ฟรี tier พอใช้ในเดือน 1-3 มั้ย

ถ้า "ใช่" ใช้ตัวนั้นก่อน ไม่ต้องสมัคร paid
ถ้า "ไม่" ดูราคา paid ของแต่ละตัว เลือกถูกสุดที่ feature ครบ

### คำถาม 2: เครื่องมือนี้ตอบโจทย์ 80% ของงานผมมั้ย

ไม่ต้องหา tool ที่ perfect 100% หา tool ที่ผ่าน 80% เริ่มใช้ พอเจอ 20% ที่ไม่ผ่าน ค่อยเสริม tool อื่น

### คำถาม 3: ชุมชนและ tutorial ภาษาไทยมีมั้ย

Tool ที่มีคน tutorial ภาษาไทยบน YouTube, มีคนถาม-ตอบในกลุ่ม Facebook เรียนรู้เร็วกว่า tool ที่ต้องไปหา tutorial ภาษาอังกฤษอย่างเดียว

ChatGPT, Claude, Canva มี community ไทยแข็งแรง
Midjourney, Runway ต้องพึ่ง resource อังกฤษเป็นหลัก

สำหรับคนที่ภาษาอังกฤษไม่แน่น เลือกตัวที่มี community ไทยก่อน

## 7.5 เมื่อไหร่ควรอัพเกรดจากฟรีเป็น paid

กฎของ AI Money Lab ไม่อัพเกรดจนกว่าจะรู้สึก "ฟรีเริ่มจำกัดการทำงานของฉันจริง"

### Signal ที่ควรอัพเกรด

**หนึ่ง ถึง limit ทุกวัน**

ถ้าคุณใช้ ChatGPT ฟรีแล้วเจอข้อความ "You've reached your limit" อย่างน้อย 3-4 วัน/สัปดาห์ = ถึงเวลา

**สอง ต้องการ feature paid เฉพาะ**

บาง tool ฟรีทำงานได้เพียง preview เช่น ElevenLabs ฟรี cuts off ที่ 10,000 ตัวอักษร ถ้าทำงาน commercial ต้อง paid

**สาม ลูกค้าจ่าย ROI กลับ**

ถ้าใช้ Canva Pro ประหยัดเวลา 10 ชั่วโมง/เดือน ทำให้รับงานเพิ่ม 2-3 ชิ้น = subscription 500 บาทคุ้มชัด

### Signal ที่ยังไม่ควรอัพเกรด

- "เพื่อนแนะนำ" → ไม่พอ
- "เห็นคนบน Twitter บอกว่าดีมาก" → ไม่พอ
- "อยากลองดู" → ไม่พอ

อัพเกรดเมื่อมีเหตุผลเชิงตัวเลข ไม่ใช่เพราะ FOMO

### Budget แนะนำตามเดือน

- **เดือน 1-3** 0 บาท ใช้ฟรีเท่านั้น
- **เดือน 4-6** 300-800 บาท (1-2 subscription ที่จำเป็นจริง)
- **เดือน 7-12** 500-1,500 บาท (ตามรายได้ที่โต)

รายจ่าย tool ไม่ควรเกิน 10% ของรายได้จาก side income เดือนนั้น

## 7.6 เช็คราคาล่าสุดก่อนสมัคร

ราคา subscription AI เปลี่ยนบ่อย บางครั้งทุก 3-6 เดือน

ตัวเลขที่ใช้ในบทนี้คือประมาณการช่วงต้นปี 2569 ก่อนสมัครจริง ให้

- เข้าเว็บหลักของ tool ดูราคาปัจจุบัน
- เทียบระดับว่า feature ที่อยากได้อยู่ในระดับไหน
- ดู annual plan ปกติถูกกว่า monthly 20-30%

### เคล็ดลับประหยัด

**หนึ่ง รอ Black Friday / ต้นปี**

หลาย tool มี promo ลด 20-50% ช่วง November-December และมกราคม ถ้าไม่เร่งใช้ทันที รอได้

**สอง ใช้ education email**

ถ้าคุณเป็นนักศึกษา (หรือมีอีเมล .ac.th / .edu) Canva, Notion, Figma, GitHub Copilot ฟรีหรือลดราคาเยอะ

**สาม Team plan หาร**

Claude Team, ChatGPT Team ราคาต่อคนถูกกว่า Pro ถ้าหาเพื่อน 3-5 คนแชร์ ราคาเหลือต่อหัวเพียง 300-400 บาท/เดือน (ระวังเงื่อนไขการใช้งานของ platform)

**สี่ ยกเลิกทันทีถ้าไม่ใช้**

ตั้ง reminder 25 วันหลังสมัคร เช็คว่าใช้จริงมั้ย ถ้าเดือนนั้นใช้น้อยกว่า 5 ครั้ง ยกเลิก

## 7.8 สรุปบทที่ 7 + Action

### สรุป

สภาพตลาด AI tools ใหญ่และเปลี่ยนเร็ว แต่คุณไม่ต้องรู้ทุกตัว

กฎ เลือก 3 tools ที่ตรงกับ path ของคุณ ใช้ให้เก่ง ฟรีก่อน

- Ch 10 (เขียน) ChatGPT + Claude + Canva
- Ch 11 (แปล) Claude + ChatGPT + NotebookLM
- Ch 12 (Affiliate) ChatGPT + Perplexity + Canva
- Ch 13 (Product) ChatGPT + Canva + Ideogram
- Ch 14 (Video) ChatGPT + ElevenLabs + CapCut

อัพเกรดเป็น paid เมื่อมีเหตุผลเชิงตัวเลข ไม่ใช่ FOMO

เดือน 1-3 งบ tool 0 บาท เดือน 4-6 ไม่เกิน 800 เดือน 7+ ตามสัดส่วนรายได้

### Action

**วันนี้ ตอนนี้**

เปิดสมุด เขียน 3 อย่าง

- path ที่ฉันเลือก
- 3 tools ที่ฉันจะใช้ในเดือน 1-3
- tool ที่ฉันสมัครไปแล้วแต่ไม่ได้ใช้จริง (และจะ cancel)
