# บทที่ 6
## Workflow — ใช้ AI เหมือนทีมเล็ก

---

## 6.1 ทำไม 1 AI ทำไม่พอ

สมมติว่า 2 คนทำงานเดียวกัน เขียนโพสต์ขายคอร์ส 1 โพสต์ พร้อมภาพประกอบ

คนแรก โยนทุกอย่างให้ ChatGPT

> "ช่วยเขียนโพสต์ขายคอร์สสอนถ่ายรูปให้หน่อย พร้อมแนะนำภาพประกอบที่เหมาะ"

ผลลัพธ์ได้ caption ทั่วไป 1 ก้อน กับคำแนะนำภาพเป็นข้อความว่า "ใช้ภาพคนถ่ายรูปตอนพระอาทิตย์ตก" เสร็จแล้วคนแรกเปิด Google หารูป ได้รูปที่ใช้ไม่ได้จริง

คนที่สอง แบ่งงานให้ AI 3 ตัว

- ChatGPT คิด outline + angle ของโพสต์
- Claude เขียน caption ตาม outline (Claude เขียนภาษาไทยลื่นกว่าในงาน creative)
- Ideogram สร้างภาพที่ตรงกับ caption

ผลลัพธ์ได้ caption ที่มี structure ดี อ่านแล้วไม่รู้สึกว่า AI เขียน ภาพที่ตรงกับ caption พร้อม post

คนที่สองเสร็จเร็วกว่า เพราะแต่ละขั้น AI ตัวที่เหมาะกับงานนั้นทำให้ ไม่ได้ฝืนให้ตัวเดียวทำทุกอย่าง

นี่คือหัวใจของบทนี้ AI 1 ตัวเก่งบางอย่าง ไม่เก่งทุกอย่าง การรวม 2-3 ตัวเข้าด้วยกันคือ workflow

---

## 6.2 แนวคิดหลัก — แต่ละ AI มี strength ต่าง

ก่อนไป workflow ต้องเข้าใจว่าใครเก่งอะไร

### ChatGPT

- เก่ง brainstorm ไอเดีย
- เก่ง outline โครงสร้าง
- เก่ง code / Excel formula
- ภาษาอังกฤษเขียนดี
- ภาษาไทยพอใช้ แต่บางทีแข็ง

### Claude

- เก่งงาน creative writing ภาษาไทย
- เก่งงานยาว (long-form)
- เก่งอ่านเอกสารยาว 50-100 หน้า
- ตาม instruction ซับซ้อนได้ดี
- ไม่มี image generator built-in

### Gemini

- เก่งค้นข้อมูลล่าสุด (เชื่อม Google Search)
- เก่งทำงาน multimodal (รูป + text)
- ฟรีตลอด (ยังไม่มี paywall หนัก)
- ภาษาไทยใช้ได้

### เครื่องมืออื่นที่จะอยู่ใน workflow

- **Canva / Canva AI** ทำ design และ template
- **Ideogram / Midjourney** สร้างภาพคุณภาพสูง
- **DeepL** แปลระหว่างภาษายุโรป
- **Grammarly / LanguageTool** ตรวจ grammar อังกฤษ
- **Perplexity** search พร้อม citation
- **NotebookLM** อ่านเอกสารหลายไฟล์พร้อมกัน

คุณไม่ต้องใช้ทุกตัว บทที่ 7 จะพูดรายละเอียดแต่ละตัว

## 6.3 Workflow 1: เขียน Content จากศูนย์ถึงปล่อย

ใช้เมื่อ ต้องเขียนบทความ blog, โพสต์ Facebook ยาว, email marketing ที่ต้องการคุณภาพสูง

### ขั้นตอน (ใช้ 3 ตัว)

**ขั้น 1: ChatGPT — Outline + Angle (5 นาที)**

```
คุณเป็นนักวางแผน content สำหรับเพจ Facebook

ช่วยวาง outline สำหรับโพสต์ขาย [สินค้า] 

บริบท
- สินค้า [ระบุ]
- กลุ่มเป้าหมาย [ระบุ]
- เป้าหมาย [ขาย / ให้ความรู้ / แชร์ประสบการณ์]

Output ที่ต้องการ
- 3 angle ที่ต่างกัน
- แต่ละ angle: ประโยคจับใจ + 3-4 ประเด็นหลัก + CTA
- บอกด้วยว่า angle ไหนน่าจะทำให้คนปิดการขายดีสุดและทำไม
```

**ขั้น 2: Claude — เขียน Caption เต็มตัว (5 นาที)**

Copy angle ที่เลือกจาก ChatGPT ส่งเข้า Claude

```
ช่วยเขียนโพสต์ Facebook ตาม outline นี้ให้เป็นเนื้อจริง

[paste outline จาก ChatGPT]

เงื่อนไข
- ภาษาไทยลื่น อ่านเหมือนคนเขียน ไม่ใช่ AI
- ความยาว 250-350 คำ
- tone เป็นกันเอง
- ห้ามใช้คำว่า "ในยุคปัจจุบัน" "อย่างไรก็ตาม" "ดังนั้น"
- จบด้วย CTA ที่ชัด
```

**ขั้น 3: ตัวคุณ + Claude — Edit ให้เป็นธรรมชาติ (5 นาที)**

อ่านออกเสียง ตรงไหนสะดุด สั่ง Claude

> "ย่อหน้า 2 อ่านแล้วขัด ลองเขียนใหม่ในโทนเดิม"

ตรงไหนที่ generic ไป

> "ใส่ตัวอย่างจริงลงในย่อหน้า 3 เช่นชื่อร้านหรือสถานที่จริง"

15 นาที บทความพร้อมปล่อย

### ทำไมไม่ใช้ ChatGPT หรือ Claude ตัวเดียว

ChatGPT วาง outline เก่ง แต่เวลาเขียนภาษาไทยยาว ๆ มักแข็งกว่า Claude
Claude เขียนลื่นแต่บางทีเริ่ม outline แล้วเดินออกนอกทาง

แบ่งงาน เร็วกว่าและดีกว่าให้ตัวเดียวทำทั้งหมด

## 6.4 Workflow 2: ภาพ + Caption สำหรับโพสต์ขาย

ใช้เมื่อ ทำโพสต์ขายที่ต้องมีภาพประกอบ เช่น เปิดตัวสินค้าใหม่, โปรโมชัน, content ที่ต้อง visual

### ขั้นตอน (ใช้ 3 ตัว)

**ขั้น 1: ChatGPT — Concept ภาพ + Caption (5 นาที)**

```
คุณเป็น creative director 

ช่วยคิด concept สำหรับโพสต์ Facebook ขาย [สินค้า]

Output ที่ต้องการ
1. Concept ภาพหลัก 1 รูป (บรรยาย 2-3 ประโยค พร้อม visual direction)
2. Caption 2 variation ความยาว 80-120 คำ
3. Hashtag 5 ตัว

บริบท [ใส่รายละเอียดสินค้าและกลุ่มเป้าหมาย]
สไตล์ [minimal / colorful / warm / cool]
```

**ขั้น 2: Ideogram หรือ Canva AI — สร้างภาพ (5-10 นาที)**

Copy "Concept ภาพหลัก" จาก ChatGPT เอามาเป็น prompt ของ image generator

Ideogram prompt ตัวอย่าง
```
A flat lay of a ceramic coffee cup with cold brew on wooden table, 
morning sunlight, minimal style, top-down view, 
warm tone, realistic photography
```

Generate 2-4 ตัวเลือก เลือกที่ชอบ

**ขั้น 3: Canva — ใส่ text + layout (5 นาที)**

เปิด Canva เลือก template "Facebook post square"
Upload ภาพจาก Ideogram
ใส่ text tagline ถ้าจำเป็น
Download เป็น PNG / JPG

15-20 นาที ได้โพสต์พร้อมภาพ

### เคล็ดลับ Prompt ภาพ

AI image generator เข้าใจ prompt แบบภาษาอังกฤษดีกว่าไทยมาก ต่อให้คุณใช้ภาพสำหรับตลาดไทย prompt ใช้อังกฤษเสมอ

Element ที่ควรระบุ
- Subject (คน/ของ/ฉาก)
- Lighting (morning, golden hour, studio light)
- Style (minimal, vintage, realistic photography)
- Camera angle (top-down, close-up, wide shot)
- Color mood (warm, cool, monochrome)

ขาด 1 element ผลลัพธ์จะกลาง ๆ ครบ element ผลลัพธ์จะมี character

## 6.5 Workflow 3: สรุป + แปลเอกสาร

ใช้เมื่อ รับงานสรุปเอกสารภาษาอังกฤษให้ลูกค้าไทย หรือแปลงานวิจัยภาษาต่างประเทศ

### ขั้นตอน (ใช้ 3 ตัว)

**ขั้น 1: Claude — สรุปเอกสารยาว (10 นาที)**

```
คุณเป็นนักวิเคราะห์ที่สรุปงานให้ผู้บริหารอ่าน 5 นาทีเข้าใจ

ช่วยสรุปเอกสารต่อไปนี้

Format ที่ต้องการ
1. TL;DR 1 ประโยค
2. Key findings 5-7 ข้อ (bullet)
3. ตัวเลขสำคัญ 3-5 ตัวพร้อมบริบท
4. Action items 3 ข้อ

ผู้อ่าน [ระบุ]
ความยาวรวม 1 หน้า A4

[paste เอกสาร 30-60 หน้า]
```

Claude รับเอกสารยาวได้ดี context window ใหญ่

**ขั้น 2: ChatGPT — Localize เป็นไทยที่ใช้ได้ (5 นาที)**

ถ้าสรุปขั้น 1 ออกมาเป็นอังกฤษ (เพราะเอกสารต้นฉบับอังกฤษ) ต้องแปลเป็นไทยที่เป็นธรรมชาติ

```
ช่วย localize สรุปนี้เป็นภาษาไทยที่เป็นธรรมชาติ 
ไม่ใช่แปลตรงตัว

เงื่อนไข
- ใช้ภาษาไทยที่ผู้บริหารเข้าใจ
- ศัพท์เฉพาะทางภาษาอังกฤษใส่ในวงเล็บ
- ปรับ tone ให้เป็นรายงานธุรกิจไทย

[paste สรุปจาก Claude]
```

**ขั้น 3: ตัวคุณ — อ่านและตรวจเทียบ (10 นาที)**

- เทียบตัวเลขทุกตัวกับต้นฉบับ
- ตรวจชื่อเฉพาะ ชื่อบริษัท ไม่มีผิด
- อ่านออกเสียง แก้จุดที่ขัด

25 นาที งานสรุป 50 หน้าเสร็จ

### ทำไมไม่ใช้ ChatGPT สรุปตั้งแต่แรก

ChatGPT รับไฟล์ยาวได้ แต่สมรรถนะการสรุปเอกสารยาวจริงจัง Claude ทำได้ consistent กว่า
ChatGPT ภาษาไทยเขียนเป็น report ลื่นกว่า Claude

แบ่งงานตาม strength แต่ละขั้นลื่น

## 6.6 เมื่อไหร่ใช้ Workflow เมื่อไหร่ใช้ AI เดียว

ไม่ใช่ทุกงานต้องรันผ่าน 3 AI

### ใช้ AI เดียวได้เมื่อ

- งานสั้น ใช้เวลาทำไม่เกิน 10 นาที (แคปชัน 1 โพสต์ / email 1 ฉบับ)
- งาน repetitive ที่มี template แล้ว
- งานที่คุณแค่ต้องการ draft เร็ว ๆ
- ลูกค้าไม่คาดหวังคุณภาพสูง

### ใช้ Workflow เมื่อ

- งานที่ลูกค้าจ่าย 1,000+ บาท
- งานที่ต้องเผยแพร่ต่อสาธารณะ (blog, ads)
- งานที่ต้อง final polish (output ของลูกค้าจริง)
- งานที่ AI ตัวเดียวทำแล้วรู้สึกว่ายังไม่ถึง

กฎง่าย ๆ ถ้าเป็นงานที่คุณใส่ชื่อไปลงท้ายได้ ใช้ workflow ถ้าไม่ ใช้ AI เดียวพอ

## 6.7 ข้อควรระวัง

### หนึ่ง AI ไม่ต่อกันอัตโนมัติ

ChatGPT ไม่ส่งข้อความเข้า Claude ให้อัตโนมัติ คุณต้อง copy-paste manual ทุกขั้น

มี tool ระดับ advanced (Zapier, Make, n8n) ที่เชื่อม AI ได้ แต่สำหรับคนเริ่ม manual copy-paste ง่ายกว่าและเสียเวลาแค่ 10 วินาทีต่อขั้น

### สอง อย่าใช้ workflow เยอะเกิน 4 ขั้น

มือใหม่ชอบคิดว่า "ยิ่งเยอะยิ่งดี" ใส่ 5-6 AI ลงใน workflow ผลคือเสียเวลามากกว่าประหยัด

3 ขั้นคือจุดที่เหมาะสุด 4 ขั้นเริ่มรู้สึกหนัก 5 ขั้นขึ้นไปควรย้อนกลับไปดูว่ามีขั้นไหนตัดได้

### สาม จดจำ context ระหว่างขั้น

ทุกครั้งที่เปลี่ยน AI ต้องใส่ context ให้ใหม่ เพราะ AI ตัวถัดไปไม่รู้ว่าตัวก่อนหน้าคุยอะไร

วิธี เก็บ note สำหรับทุก workflow บันทึกไว้ว่า
- Goal ของโปรเจกต์
- ข้อมูลสินค้า/ลูกค้า
- Output ของแต่ละขั้น

พอเริ่มขั้นต่อไป copy note นี้ใส่เป็น context ให้ AI เลย

## 6.8 Template Workflow พร้อมใช้ 3 ชุด

Copy ไปเก็บไว้ เปลี่ยน [ช่องว่าง] ตามงาน

### Template A — Content สำหรับ Social

```
Goal: เขียนโพสต์ Facebook ขาย [สินค้า]

ขั้น 1 ChatGPT — Outline
Prompt: คุณเป็นนักวางแผน content ช่วย outline 3 angle 
สำหรับโพสต์ขาย [สินค้า] กลุ่มเป้าหมาย [ระบุ]

ขั้น 2 Claude — Caption
Prompt: ช่วยเขียน caption ตาม outline นี้ [paste] 
ภาษาไทยลื่น 250-300 คำ tone [ระบุ]

ขั้น 3 Self-edit (15 นาที)
- อ่านออกเสียง แก้จุดขัด
- ใส่ตัวอย่างจริง
- ตรวจ CTA

เวลารวม 20-25 นาที
```

### Template B — โพสต์ที่มีภาพ

```
Goal: โพสต์ Facebook พร้อมภาพประกอบสำหรับ [สินค้า/ประกาศ]

ขั้น 1 ChatGPT — Concept + Caption + Translate prompt
Prompt: คุณเป็น creative director ช่วยคิด concept ภาพ 
+ caption 2 variation สำหรับ [สินค้า]

ก่อนใส่ concept ให้ image generator ให้ใช้ mini-prompt นี้
ใน ChatGPT ต่อทันที "ช่วยแปล concept นี้เป็น image 
generation prompt ภาษาอังกฤษ ครบทุก element (subject, 
lighting, style, camera angle, color mood): [ใส่ concept ไทย]"
AI จะแปลให้ใช้ใส่ image generator ได้ทันที

ขั้น 2 Ideogram / Canva AI — สร้างภาพ
Prompt: [paste prompt อังกฤษจากขั้น 1]

ขั้น 3 Canva — Finalize
- Upload ภาพ
- ใส่ tagline ถ้าต้องการ
- Export 1080x1080

เวลารวม 20-30 นาที
```

### Template C — สรุปเอกสารภาษาอังกฤษเป็นไทย

```
Goal: สรุปเอกสาร [ชื่อ] 30-60 หน้าเป็น executive summary ไทย 1-2 หน้า

ขั้น 1 Claude — สรุปเป็นอังกฤษ
Prompt: ช่วยสรุปเอกสารนี้เป็น TL;DR + 5-7 key findings 
+ ตัวเลขสำคัญ [paste เอกสาร]

ขั้น 2 ChatGPT — Localize เป็นไทย
Prompt: ช่วย localize สรุปนี้เป็นไทยธรรมชาติ 
ศัพท์เทคนิคใส่วงเล็บ [paste สรุปจากขั้น 1]

ขั้น 3 Self-verify (10 นาที)
- เทียบตัวเลขกับต้นฉบับ
- ตรวจชื่อเฉพาะ

เวลารวม 25-35 นาที
```

## 6.9 สรุปบทที่ 6 + Action

### สรุป

AI 1 ตัวเก่งบางอย่าง Workflow คือการใช้ strength ของแต่ละตัวรวมกัน

ChatGPT เก่ง brainstorm + outline
Claude เก่ง creative writing + เอกสารยาว
Image generator เก่งทำภาพ
Canva finalize layout

3 ขั้นคือจุดที่เหมาะสุด 4 ขั้นเริ่มหนัก 5+ ขั้นควรตัด

ใช้ workflow เมื่องานจริงจัง ไม่ใช้ทุกงาน งานสั้นใช้ AI เดียวพอ

### Action

**วันนี้**

ลอง Workflow A (content สำหรับ social) กับสินค้าจริงหรือสมมติ 1 ชิ้น

- ขั้น 1 ChatGPT วาง outline (5 นาที)
- ขั้น 2 Claude เขียน caption (5 นาที)
- ขั้น 3 อ่านออกเสียง edit (5 นาที)

รวม 15 นาที เทียบกับเวลาที่ใช้ตอนให้ AI เดียวทำทั้งหมด แล้วเทียบคุณภาพ

ถ้าเห็นความต่างชัด คุณเข้าใจ workflow แล้ว
