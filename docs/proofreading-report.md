## 🟠 Important Issues (ควรแก้ก่อน publish)

### 7. Typos / spelling

**Ch 1.6 (line 140): "สรรพตำรา"** — คำนี้ไม่มีในพจนานุกรม
> "**ไม่ใช่สรรพตำรา AI** ผมไม่พูดเรื่อง AGI..."

แนะนำ: "ไม่ใช่ตำรา AI ฉบับครบทุกเรื่อง" หรือ "ไม่ใช่สารานุกรม AI"

**Ch 2.1 (line 18): "พึ่งเริ่ม"** ผิด ควรเป็น "**เพิ่งเริ่ม**"
> "แต่ถ้าคุณ**พึ่งเริ่ม** การสมัคร 3 ตัวพร้อมกัน..."
"พึ่ง" = พึ่งพา, "เพิ่ง" = just (เวลา) → ในบริบทนี้คือ "just started" = เพิ่งเริ่ม

(ที่อื่นใช้ "เพิ่ง" ถูกต้องแล้ว เช่น Ch 1.1 "เพิ่งกล่อมลูก", Ch 3.1 "เพิ่งสมัคร Claude", Ch 4.1 "เพิ่งรู้จัก ChatGPT")

**"พึ่งได้"** — คำนี้คลุมเครือ (ใช้หลายที่: Ch 1.3, Ch 12.1, Ch 14.1, Ch 15.7) หมายถึง "พึ่งพาได้" = rely on → ถูกต้อง grammatical แต่บางคนอาจสับสนกับ "เพิ่งได้" **ถ้ายืนยันว่าหมายถึง "reliable income"** แนะนำเปลี่ยนเป็น "**พึ่งพาได้**" ให้ชัดกว่า

### 8. AI tells — sample ส่วนใหญ่สะอาดมาก (strong)

ตรวจ "ดังนั้น / อย่างไรก็ตาม / ในยุคปัจจุบัน" — ไม่พบใน body text เลย ใช้เฉพาะใน prompt template ที่สั่งห้าม และใน Ch 8 ที่พูดถึง AI tells เอง → **excellent**

**ยังเหลือจุดที่ย้อนกลับมาเช็ค:**
- **Em dash (—) ใช้ในหัวข้อ "บทต่อไป" ทุกบท** (line สุดท้าย 16/16 บท) — เป็น structural pattern ที่ AI สร้างโดยธรรมชาติ คนอ่านอาจรู้สึกว่าเป็น template ถ้าเห็นทุกบท
