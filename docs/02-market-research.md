# 02. ตลาดและคู่แข่ง

สิ่งที่สำรวจคือคู่แข่งขาย e-book สอน AI / ChatGPT ในไทย ทั้ง digital product ที่ขายตรงกับคอร์สออนไลน์ที่ชนงบลูกค้าเรา

วิธีสำรวจ: ค้นหาเว็บด้วยหลายคีย์เวิร์ด + ตามลิงก์ไปเปิด landing page ตรง (ที่เปิดได้) รวบรวมราคา ข้อเสนอ positioning ความเห็นผู้ซื้อ

## ภาพรวมตลาด

ตลาด e-book / คอร์สสอน AI ในไทยแบ่งได้ 3 กลุ่มหลัก

**กลุ่ม 1: E-book กับ Prompt Pack** คู่แข่งตรงของเรา ราคา 189-399 บาท เนื้อหาเบา เน้น prompt สำเร็จรูป ไม่ได้สอนพื้นฐานมาก ตัวอย่าง Data-Espresso, Benzio Prompt Pack, Pintobook

**กลุ่ม 2: คอร์สออนไลน์เต็มรูป** ราคา 790-3,000+ บาท video + worksheet สอนเต็ม ตัวอย่าง Benzio GPT, SkillLane, Skooldio, FutureSkill

**กลุ่ม 3: คอร์สฟรีของมหาวิทยาลัย** ฟรีหมด วิชาการ เนิบ ไม่ตอบโจทย์คนหารายได้ Chula MOOC, KU MOOC, SET e-Learning

## คู่แข่งตรงแต่ละเจ้า

### Data-Espresso — คู่แข่งหลักสุด

มี e-book 3 เล่ม (HR, บริหาร, รายได้หลักแสน) ราคาเต็ม 299-399 ลดเหลือ 189-219 ประจำ

**Positioning** "Prompt สำเร็จรูป ใช้ง่าย ได้ผลจริง"

จุดแข็ง: เจาะ niche ชัด แบรนด์เริ่มติด ลดราคาเก่ง

จุดอ่อน: เนื้อหาแค่ prompt pack ไม่ได้สอนพื้นฐาน ใครที่ไม่เคยใช้ ChatGPT มาก่อน ซื้อมาแล้วเปิดอ่านไม่เข้าใจว่า prompt ใช้ที่ไหน + ขาดแพ็กเกจ "ของแถมเยอะ" ขายแค่ PDF เปล่า

**ของเราต่างยังไง** เราสอนตั้งแต่ 0 ก่อน prompt เป็นส่วนหนึ่งใน Part 2 ไม่ใช่ตัว product หลัก และเรามี Starter Kit เพิ่ม

### Benzio — eBook รวม 1,000+ prompts

ราคาประมาณ 290 บาท positioning "ให้เยอะ คุ้ม"

จุดแข็ง: brand อาจารย์แน่น, promotion เก่ง

จุดอ่อน: 1,000 prompts มือใหม่เจอแล้วงงว่าควรเริ่มที่ไหน ตกอยู่ใน paradox of choice

**ของเราต่างยังไง** เราไม่ได้พยายามให้เยอะที่สุด ให้ prompt ที่เหมาะกับ "วิธีหาเงิน" ที่สอนโดยตรง แต่ละ prompt มีที่ใช้ชัด

### Pintobook — 100 Marketing ChatGPT Prompt

Niche แคบเฉพาะ marketing ไม่ใช่คู่แข่งตรงเท่าไหร่ แต่อาจดูดคนกลุ่มเดียวกัน

## คู่แข่งอ้อม

### หนังสือกายภาพ "ใช้ AI ทำเงิน ทำงานแทน เพิ่มรายได้ 100 เท่า"

ขายที่นายอินทร์ Se-Ed ศูนย์หนังสือจุฬาฯ โชว์ว่าขายดีอันดับ 1 ในหมวด AI ของไทย

ตรงกับ angle ที่เราเลือก (หารายได้ด้วย AI) แต่:
- เป็นหนังสือเล่มกายภาพ ซื้อยากกว่า
- ไม่มี community / update ต่อเนื่อง
- ไม่มีส่วนลด อะไรมากมาย

เราไม่ชนตรงเพราะเป็น digital product + มี community + ราคาต่างกัน

### 7D Academy, WePlus Academy, Skooldio, SkillLane

คอร์สสอนใช้ AI ทำ content / ebook ขาย ราคาหลักพันถึงหลักหมื่น

คนงบเยอะซื้อคอร์สไปก่อนเรา เราได้คนงบกลาง (349) ที่ยังไม่พร้อมจ่ายหลักพัน ซึ่งตลาดนี้ใหญ่พอ

## ช่องว่างในตลาด (ที่เราจะยึด)

ดูคู่แข่งทั้งหมดแล้วเจอ 4 ช่องที่ไม่มีใครครอง

**1. ไม่มีใครเป็น "คู่มือครบ จาก 0 ถึงใช้งาน"**
ทุกเจ้าเป็น prompt pack (รวมคำสั่ง) หรือคอร์สราคาพัน ไม่มี e-book ราคากลาง ที่สอนตั้งแต่พื้นฐาน

**2. ไม่มีใครใช้ angle "ภาษาคน ไม่มีศัพท์เทค"**
คู่แข่งทุกเจ้าเล่นคำ "Prompt Engineering", "Generative AI", "AI Agent" — คน 35+ / แม่บ้านเห็นแล้วกลัว ปิดเพจ

เราจะขายด้วยภาษาแบบ "ให้ AI ทำงานแทน", "เริ่มได้ในวันเดียว", "ไม่ต้องเก่งเทค"

**3. ไม่มีใคร bundle "ebook + template + community"**
ทุกเจ้าขายแค่ PDF เปล่า หรือคอร์สที่ 1,000+ เราทำ package ที่ 349 ให้ perceived value สูงสุด

**4. ไม่มีใครทำ community-driven funnel**
ทุกเจ้ายิง ad → หน้าขาย → จบ เราดึงเข้า community ก่อน (LINE OpenChat + FB Group) ทำให้ conversion ดี + เตรียม audience ไว้ขายสินค้าถัดไป

## ความเสี่ยงที่ต้องระวัง

**Data-Espresso ลดราคาเหลือ 219 ประจำ**
เราขาย 349 ต้องโชว์ให้เห็นชัดว่า "ของเยอะกว่า" ไม่ใช่แค่ "แพงกว่าเพราะคุณภาพ" ต้องเห็นเป็นรูปธรรม

**คนคิดว่า e-book เดี๋ยวนี้ AI สร้างเองได้**
ต้องมี authority signal ให้เห็น — ชื่อผู้เขียน ประสบการณ์ case study จริง screenshot ผลงาน

**คอร์สฟรีของจุฬา / KU MOOC**
ดูดคนที่ขยันเรียน เราต้องขายที่ "ประหยัดเวลา ไม่ต้องฟัง lecture 20 ชั่วโมง" + outcome-driven มากกว่าวิชาการ

## Positioning ของเรา (สรุป 1 บรรทัด)

**"คู่มือ AI ฉบับภาษาคน เริ่มจาก 0 พร้อม Starter Kit ใช้งานได้ภายในวันเดียว"**
